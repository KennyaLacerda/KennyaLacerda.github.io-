Instruções para Iniciar um Servidor Django

  1. Instalar o Django: Certifique-se de ter o Django instalado. Você pode instalá-lo usando o seguinte comando:
    pip install django
  2. Criar um Projeto Django: Abra o terminal e execute o seguinte comando, substituindo `nome_do_projeto` pelo nome desejado para o seu projeto:
    django-admin startproject nome_do_projeto
  3. Navegar até o Diretório do Projeto: Mude para o diretório do projeto com o seguinte comando:
    cd nome_do_projeto
  4. Criar uma Aplicação: O Django é baseado no conceito de "aplicações". Crie uma aplicação dentro do seu projeto com o seguinte comando, substituindo `nome_da_aplicacao` pelo nome desejado:
    python manage.py startapp nome_da_aplicacao
  5. Migrações e Aplicação: Execute o seguinte comando para aplicar as migrações iniciais e criar o banco de dados SQLite (ou o motor de banco de dados configurado em seu arquivo `settings.py`):
    python manage.py migrate
  6. Criar um Superusuário (opcional): Você pode criar um superusuário para acessar o painel de administração do Django. Isso é opcional, mas pode ser útil:
    python manage.py createsuperuser
  7. Iniciar o Servidor: Finalmente, inicie o servidor de desenvolvimento executando o seguinte comando:
    python manage.py runserver
  8. Acessar a Aplicação: Abra seu navegador e visite http://127.0.0.1:8000/. Você também pode acessar o painel de administração em http://127.0.0.1:8000/admin/ usando as credenciais do superusuário que você criou.

Lembre-se de que o servidor de desenvolvimento do Django não deve ser usado em produção. Para implementações em produção, é recomendável usar um servidor web como Nginx ou Apache e configurar o Django para uso em produção.
